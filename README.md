# my_pacman
Pac-Man game written in C, using the ncurses.



**This is an old student project, it is not maintained or supported.**

## Download
### GNU/Linux
```bash
git clone https://github.com/Sad-theFaceless/my_pacman.git
```
#### Dependencies
- lib ncurses

## How to use
### Launch the game
```bash
./pacman.sh
```
### Reset the high score
```bash
./delete_highscore.sh
```
